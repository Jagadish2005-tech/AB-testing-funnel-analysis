# A/B Testing & Funnel Analysis System

## 📌 Overview

This project analyzes user behavior in an e-commerce platform to evaluate conversion performance using A/B testing and funnel analysis. It simulates real-world experimentation and decision-making workflows used in product analytics.

---

## 🎯 Objectives

* Compare conversion rates between Variant A and B
* Identify drop-off points in user journey
* Perform statistical testing to validate results
* Build a dashboard for business insights

---

## 🛠️ Tech Stack

* Python (Pandas, NumPy, Statsmodels, Dask)
* SQL (SQLite)
* Power BI

---

## 📊 Key Analysis

### 🔹 Funnel Analysis

Tracked user journey across:
View → Cart → Purchase
Identified major drop-offs between stages.

---

### 🔹 A/B Testing

* Compared conversion rates between variants
* Applied statistical hypothesis testing (p-value analysis)
* Result: No statistically significant difference (p = 0.13)

---

## 📈 Dashboard

Built an interactive Power BI dashboard including:

* Conversion rate comparison (A vs B)
* Funnel visualization
* KPI metrics and statistical insights

---

## ⚙️ Automation

* Designed a Python-based pipeline to process and analyze data
* Scheduled execution using Task Scheduler for automated runs
* Stored results in SQLite database for consistent reporting

---

## 🚀 Key Insights

* Both variants perform similarly in terms of conversion
* Significant drop observed between view and cart stage
* Opportunity to optimize user experience in early funnel stages

---

## 📌 Conclusion

This project demonstrates how A/B testing and funnel analysis can be used to support data-driven decision making in real-world scenarios.
